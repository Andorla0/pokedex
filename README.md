# 🧩 Pokedex App - Nivel 1

Aplicación básica de Pokedex construida con React.  
Este nivel incluye listado de Pokémon, filtros y búsqueda.

## Tecnologías

- React (Vite)
- Tailwind CSS
- PrimeReact
- React Query (TanStack Query)
- PokeAPI

## ✨ Funcionalidades

- Listado de Pokémon
- Búsqueda por nombre
- Filtro por tipo
- Skeleton loading 
- Diseño responsive básico
